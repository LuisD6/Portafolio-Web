# Mi Portafolio

Este repositorio contiene el código fuente de mi portafolio personal, en donde muestro mis proyectos, habilidades, las herramientas que utilizo y mis datos de contacto. 

El proyecto fue desarrollado utilizando las siguientes tecnologías:
- **Next.js**
- **React**
- **JavaScript**
- **Tailwind CSS**
- **Framer Motion**
- **React Icons**

---

Este proyecto fue creado con [Next.js](https://nextjs.org) y fue iniciado utilizando [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).
